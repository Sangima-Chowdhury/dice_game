# Dice Game

A simple Python command-line dice rolling game.

## Features
- Roll two dice
- Generate random numbers
- Display dice results
- Calculate total
- Continue or exit based on user input

## Technologies Used
- Python
- Random module
- Git & GitHub

## What I Learned
This project helped me practise Python fundamentals including loops, conditionals, user input, random number generation, and simple program flow.
